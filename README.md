# learn-typescript
learn [typescript](https://www.typescriptlang.org/) records

## How to start project?
1.  install dependence
```bash
npm install
# or
pnpm install
```
2.  initial `tsconfig.json` config file
```bash
npx tsc --init
```
3.  run the follows command in the terminal
```bash
ts-node xxx.ts
```
